Landsat MSS satellite sensor spectral calibration evaluation using time series graphed with Plotly.js API

Each point in the graphs represents the median pixel value for all non-cloud pixels in an image. Images include dates throughout the year so that phenology is expressed. Also shown is the median of the medians for each sensor as a line to assess offset. There are three spectral indices shown: tasseled cap brightness, greenness, and wetness. The Plotly graphs are interactive - use the mouse wheel to zoom in and out and see the Ploty icons in the upper-right of the figures for more options.

Tip: Click the buttons in the upper-left to load a different scene's data.
