Landsat MSS satellite image time series graphed using Plotly

Each point in the graphs represents the median pixel value for all non-cloud pixels. Images include dates throughout the year so that phenology is expressed. There are three spectral indices shown: tasseled cap brightness, greenness, and wetness. The Plotly graphs are interactive - use the mouse wheel to zoom in and out and see the Ploty icons in the upper right of the figures for more options.
